nengolib.PerfectLIF¶

class nengolib.PerfectLIF(tau_rc=0.02, tau_ref=0.002, min_voltage=0, amplitude=1)[source]¶

Spiking version of the leaky integrate-and-fire (LIF) neuron model.

This model spikes with the same rate as nengo.LIFRate assuming zero-order hold (ZOH). [1]

Parameters:

Additional arguments passed to nengo.LIF.

**kwargs : dictionary, optional

Additional keyword arguments passed to nengo.LIF.

See also

nengo.LIF, nengo.neurons.NeuronType

Notes

This is Nengo’s default neuron model since nengo>=2.1.1.
