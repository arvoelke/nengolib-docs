nengolib.signal.balred¶

nengolib.signal.balred(sys, order, method='del')[source]¶

Reduces a linear system to given order using balance and modred.

Parameters:
sys : linear_system_like

Linear system representation.

order : integer

Number of dimensions to keep.

method : string, optional

Model order reduction method passed to modred().
Returns:
:class:`.LinearSystem`

Balanced and reduced linear system in state-space form.

See also

balance(), modred()

References

[1]https://www.mathworks.com/help/control/ref/balred.html

See Linear System Model Reduction for a notebook example.
